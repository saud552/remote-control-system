## 🔧 **كيفية الاستخدام**

### 🚀 **تشغيل البوت:**
```bash
# في بيئة Conda
conda activate tf_env
python run_bot_conda.py
```

### 📱 **اختبار الأزرار:**
1. أرسل `/start` للبوت
2. اضغط على أي زر رئيسي
3. اختبر الأزرار الفرعية
4. تأكد من الاستجابة الصحيحة
