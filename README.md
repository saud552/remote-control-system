# نظام التحكم عن بعد المتقدم v2.1.5

## 🚀 نظرة عامة

نظام تحكم عن بعد متقدم ومتطور يوفر إمكانية التحكم في الأجهزة عن بعد مع ميزات أمنية متقدمة وأداء محسن.

## ✨ الميزات الرئيسية

### 🔧 التحكم المتقدم
- تحكم كامل في الأجهزة عن بعد
- إدارة متقدمة للأجهزة
- أوامر متطورة ومخصصة
- مراقبة في الوقت الفعلي

### 🛡️ الأمان المتطور
- تشفير AES-256-CBC للبيانات
- حماية من الهجمات والـ DDoS
- Rate Limiting ذكي
- مراقبة النشاط المشبوه

### ⚡ الأداء المحسن
- مراقبة الأداء في الوقت الفعلي
- تحسين الذاكرة تلقائياً
- تنظيف البيانات القديمة
- استجابة سريعة ومستقرة

### 🌐 الواجهات المتعددة
- واجهة ويب متطورة
- بوت تيليجرام متقدم
- API RESTful
- WebSocket للاتصال المباشر

## 🛠️ متطلبات النظام

- **Node.js** >= 16.0.0
- **npm** >= 8.0.0
- **Python3** >= 3.9 (للبوت)
- **Git** (للنشر)

## 📦 التثبيت والتشغيل

### 1. تثبيت التبعيات

```bash
# تثبيت تبعيات خادم الأوامر
cd remote-control-system/command-server
npm install

# تثبيت تبعيات واجهة الويب
cd ../web-interface
npm install

# تثبيت تبعيات بوت تيليجرام
cd ../telegram-bot
pip3 install -r requirements.txt
```

### 2. تشغيل النظام

#### التشغيل المحلي
```bash
# من المجلد الرئيسي
cd remote-control-system
./start.sh
```

#### التشغيل على Render (السحابة)
```bash
# 1. رفع الكود إلى GitHub
git add .
git commit -m "Deploy to Render"
git push origin main

# 2. ربط المشروع بـ Render
# - اذهب إلى render.com
# - أنشئ خدمة جديدة من GitHub
# - اختر "Web Service"
# - اربط المستودع

# 3. إعداد متغيرات البيئة في Render
TELEGRAM_BOT_TOKEN=your_bot_token_here
OWNER_USER_ID=your_user_id_here
NODE_ENV=production
```

### 3. الوصول للنظام

#### المحلي
- **واجهة الويب**: http://localhost:3000
- **خادم الأوامر**: http://localhost:10001
- **بوت تيليجرام**: يعمل في الخلفية

#### على Render
- **واجهة الويب**: https://remote-control-web-interface.onrender.com
- **خادم الأوامر**: https://remote-control-command-server.onrender.com
- **نقطة فحص الصحة**: /health

## 🔍 استكشاف الأخطاء

### مشاكل شائعة وحلولها

1. **خطأ في الاتصال بالخادم**
   - تأكد من تشغيل خادم الأوامر على المنفذ 10001
   - تحقق من إعدادات Firewall
   - راجع سجلات الاتصال

2. **مشكلة في التشفير**
   - تأكد من استخدام Node.js >= 16.0.0
   - تحقق من تثبيت التبعيات بشكل صحيح
   - راجع إعدادات التشفير

3. **مشكلة في البوت**
   - تأكد من تعيين `TELEGRAM_BOT_TOKEN`
   - تحقق من اتصال الإنترنت
   - راجع سجلات البوت

4. **مشكلة في الأداء**
   - راجع استخدام الذاكرة
   - تحقق من السجلات
   - استخدم أدوات المراقبة
